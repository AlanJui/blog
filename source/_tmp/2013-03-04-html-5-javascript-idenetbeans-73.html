前陣子，研究 CakePHP 時，順便花了點時間，研究免費的 IDE 主流工具，那一個對 CakePHP Framework 的支援最好？於是又把好久沒用的 NetBeans ，拿出來用一用，這才發覺 NetBeans 7.2.1 ，已提供 CakePHP Plugin ，有著極佳的整合支援。

因為 NetBeans 對 CakePHP 極具親和力的支援，所以後來的 PHP 研究，又'開始以 NetBeans 為主。沒想到，不到兩週，NetBeans 又推出新版本 7.3 （NetBeans下載處）。

新版的 NetBeans 7.3 有著什麼樣的好功能？

以下是官網所發佈的新版功能簡介影片《NetBeans IDE 7.3 Overview》：

NetBeans IDE 7.3 Overview

 

新版功能摘要

  • 網頁編輯的結果可立即顯現：與 Google Chrome 瀏覽器結合，使輸入的 HTML Tag / CSS 設定 / JavaScript 程式碼，只要執行儲存功能，毋需手動執行網頁 Reload ，就能立即自瀏覽器看到網頁的輸出結果。網頁的編輯工作，可以像使用 DreamWeaver 一樣直覺、輕鬆； 【Fig 1】
  • 自動化的程式編輯檢查工具：撰寫程式的時候，最怕打錯字，使得 Script 程式無法正執行。操作者在編撰程式碼的過程，會將同樣的變數，以黃色底標示，程式設計人員在撰碼的過程，可透過此特性，以目視比對方法，確認變數的名稱前後一致，檢查沒有打字錯誤的問題；【Fig 2】
  • 豐富的 JavaScript 除錯輔助：可以在 NetBeans 中，對 JavaScript 設定中斷點、觀察變數內容值，直接進行'除錯工作；更能進行單元測試 (Unit Test) ，確保每個 Method/Function 可以正確執行。 【Fig 3】


【Fig 1】雖不似DreamWeaver，但也可以立即觀看TML/CSS/JavaScript對網頁的影響


【Fig 2】同樣的變數，以黃底色標示，有利於目視檢查，快速確認打字無錯誤


 【Fig 3】JavaScript 可設中斷點，觀察變數內容值

新版功能操作指引教材

想要快速了解 NetBeans 7.3 版的新功能，NetBeans 的官網提供了很棒的指引教材，讓您可以透過實際操作，親身體會。

  • Getting Started with HTML5 Applications：見識NetBeans 7.3，如何支援 HTML 5，其 Chrome with NetBeans Integration ，如何讓 HTML 網頁的編製及預覽工作，變得簡化；
  • Debugging and Testing JavaScript in an HTML5 Application：如何運用NetBeans 7.3，對 JavaScript 進行除錯；如何直接在 NetBeans 設定中斷點；觀察變數內容。如何執行 JavaScript 的單元測試 (Unit Test) 。
