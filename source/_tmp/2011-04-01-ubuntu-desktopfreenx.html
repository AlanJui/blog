有時候想偷懶，貪圖 GUI 在操作方面的便利性，所以選擇 Ubuntu Desktop 版本，當作 Ubuntu Server ，用來當作伺服器使用。

又有時，基於某些因素，我需要在一台 Windows 7 的電腦，對那台 Ubuntu Desktop 伺服器做「遙端遙控」。

雖然說，在 Ubuntu Desktop 10.10 內建「遠端桌面」功能。可是，這個遠端桌面，可不是我們在 Windows 環境所用的那個遠端桌面，所以.......，我無法使用 Windows 7 內附的「遠端桌面」來遙控那台 Ubuntu Desktop 伺服器。

Ubuntu 的行家知道， Ubuntu Desktop 10.10 內建「遠端桌面」功能，說穿了就是「VNC Server」。所以，我只需在 Windows 7 的電腦安裝 VNC Client 軟體 - RealVNC ，就能達到遠端遙控的目的。可是，RealVNC 可在 Windows 7 執行的版本，屬於要收費的版本。......................  @_@

如果需要免費的，以下的這個 FreeNX 便是福音囉！
